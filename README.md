# OOP_labs
Лабораторные работы по объектно-ориентированному программированию на c++

Алгоритм запуска программы:
1. Запускаем devcontainer через vscode
2. Выполнить сборку:
   ```
   mkdir -p build
   cd build
   cmake ..
   make
   ```
4. Запуск программы ```./labN/labN_exe```
5. Запуск тестов ```./lab1/tests```
